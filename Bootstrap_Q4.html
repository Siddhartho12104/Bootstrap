Telegram

Telegram is a cloud-based mobile and desktop messaging app with a focus on security and speed. Users can send messages and exchange photos, videos, stickers and files of any type. Telegram also provides optional end-to-end encrypted messaging with self-destruct timers, but Telegram's implementation of these features has been contested by security researchers and cryptography experts.
